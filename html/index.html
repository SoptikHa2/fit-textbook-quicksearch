Make sure you are authenticated and have access to the textbook. Log into Courses.

Textbook is searched using grep with regex. Do not mistype. Search terms separated by space are ORed. This is open source.

Important: Textbooks that are protected by CTU SSO will NOT work on Google Chrome and other Chromium-based browsers. Use Firefox. KAM textbooks work everywhere.
